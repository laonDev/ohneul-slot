## 상세 설명
```
점심마다 "뭐 먹지?" 고민하는 당신을 위한 결정 슬롯!
카테고리(한식·중식·양식·일식·분식·혼밥·회식)를 고르고 돌리기만 하면,
슬롯머신이 5초 만에 오늘의 메뉴를 골라줍니다.

• 슬롯 3릴이 후보 3개를 보여주고, 그중 하나가 당첨!
• 최근 먹은 메뉴는 덜 나오는 똑똑한 추첨 — 같은 메뉴 반복 걱정 끝
• 단골 메뉴 즐겨찾기, 이번 주 점심 기록 자동 정리
• 결과를 친구와 공유하고 같이 정하기

매일 반복되는 점심 결정 피로, 즐거운 슬롯 한 판으로 끝내세요.
```
